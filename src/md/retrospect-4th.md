# 네 번째 회고

이번 주에는 **반응형 레이아웃**을 중심으로 공부했다.
특히 **Flex를 사용한 UI 구성**이 인상 깊었다.

화면 크기에 따라 자연스럽게 변하는 레이아웃을 직접 만들어보면서,
“아, 이래서 반응형이 중요한 거구나”라는 생각이 들 정도로
반응형 레이아웃 자체에 큰 흥미를 느끼게 되었다.
단순히 화면을 맞추는 게 아니라,
사용자 환경에 맞춰 UI가 살아 움직이는 느낌이 들어서 재미있었다.

---

## Flex를 사용한 UI 만들기

Flex를 활용하면서
정렬, 간격, 방향을 의도대로 제어하는 연습을 많이 했다.

- 요소를 가로로 배치하는 것
- 중앙 정렬을 자연스럽게 만드는 것
- 화면 크기에 따라 유연하게 변하는 구조 만들기

아직은 속성을 외우듯 사용하는 단계이지만,
여러 UI를 직접 만들어보면서 점점 감이 잡히는 느낌도 들었다.

---

## Grid에 대한 첫 인상

이번 주에는 **Grid**도 조금 맛보기로 배웠다.
처음엔 표 같은 느낌이 강했는데,
막상 써보니 **굉장히 유연한 레이아웃 도구**라는 인상을 받았다.

행과 열을 기준으로 구조를 잡다 보니
복잡한 레이아웃도 훨씬 명확하게 설계할 수 있을 것 같았다.
아직은 실습도 부족하고, 제대로 활용하지는 못했지만
그래서 오히려 **Grid를 더 깊게 공부해보고 싶다**는 욕심이 생겼다.

---

## 느낀 점

아직 실력도 부족하고,
실습하면서 막히는 부분도 많지만
그럼에도 불구하고 **계속 더 해보고 싶다는 마음**이 크다.

Flex와 Grid를 자유롭게 다룰 수 있다면
훨씬 완성도 높은 UI를 만들 수 있을 것 같고,
그 과정 자체가 재미있을 것 같다는 기대도 든다.

다음 주에는
더 많이 만들어보고, 더 많이 부딪혀보면서
레이아웃에 대한 이해를 한 단계 더 끌어올리고 싶다.
