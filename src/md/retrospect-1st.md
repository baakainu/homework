# 첫 번째 회고

## 1. 회고록 개요

이번 주는 개발을 막 시작한
**완전 초보**로서
새로운 것들을
많이 배우며 보낸 시간이었다.

## 2. 배운 내용과 회고

### 2.1 CLI(Command Line Interface)

- 명령어를 입력하면 바로 결과가 출력되는 구조가 **게임 같아서 재미있었다.**
- 처음에는 `cd`, `ls`, `mkdir` 같은 기본 명령어조차 헷갈렸지만  
  반복하다 보니 *조금씩 자연스러워지는 느낌*이었다.

---

### 2.2 GIT 버전 관리

- 저장소를 생성해두고,
  작업을 진행하다가,
  과거 버전으로 이동하고,
  수정된 미래로 이어지는게
  **타임머신** 같아서 재밌었다.

```markdown
1. `git init`을 배움
2. `git add`, `git commit`을 반복하며 익숙해짐
3. `git log`로 과거 기록을 보는 것이 흥미로움
4. `git checkout` 으로 과거로 돌아갈 때 **시간 여행 느낌**🎬
```

---

### 2.3 Node.js

Node.js도 조금 배웠지만,

- 개념 이해가 아직 어려움
- **“아 이제 좀 감이 온다!”** 하는 느낌까지는 아직 먼 것 같음
- 이후 강의에서 지금 강의 내용을 리마인드 해야될 필요를 느낌

## 3. 앞으로의 목표

```markdown
나는 *음악 관련 플러그인*을 직접 개발해보고 싶다.
```

음악 작업을 오래 해오면서 자연스럽게 생긴 꿈이다.

- 구상 중인 플러그인도 있다
- 개발 과정 자체가 **매우 재미있을 것 같다**
- 취업도 목표지만, 장기적으로는 **음악과 개발을 연결**하고 싶다
